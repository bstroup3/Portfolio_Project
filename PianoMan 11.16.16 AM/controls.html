Controls


Piano -> Click/Press the keys using either the keyboard or mouse


Shift -> Displays which key on keyboard is which note


Major/Minor -> Click the switches at the bottom or use the arrow keys

to turn the notes into the major/minor chords


TextBox Feature -> Type in the name of the note you want to played.

Make sure to use capital letters the notes and add "b" for flats (eg G3 or Gb3)

and place a space between notes and place a period where you would like a pause.


Game -> Guitar Hero like game that shows the keys to play before playing them.

Try your best! Its difficult


Menu -> Looks like you found it! Use the menu to learn the

controls or learn more about the developers!
